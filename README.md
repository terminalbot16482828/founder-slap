# Founder Slap

A fun hackathon MVP: paste a messy thought, get a focused action plan + one motivational roast.

## Stack
- Frontend: React (Vite)
- Backend: Node + Express

## Quick Start

### 1) Backend
```bash
cd backend
npm install
npm run dev
```
Runs on `http://localhost:8787`.

### 2) Frontend
```bash
cd ../frontend
npm install
npm run dev
```
Open the printed Vite URL (usually `http://localhost:5173`).

## Demo Input

> I need to launch something this week, fix onboarding, and I keep doomscrolling instead of finishing the landing page.

You should get:
- Top 3 priorities
- First 10-minute step
- A roast line

## Notes
- This MVP intentionally uses deterministic/rule-based planning (no external AI API).
- “Voice note in” is represented by text input for now.
